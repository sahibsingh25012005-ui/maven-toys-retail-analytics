# 📊 Maven Toys Retail Analytics

## End-to-End Retail Data Analytics Project using SQL, Python & Power BI

> Transforming retail sales and inventory data into actionable business insights using SQL, Python, and Power BI.

## 📌 Project Overview

This project demonstrates a complete retail analytics workflow using **SQL**, **Python**, and **Power BI**. It answers **18 business questions**:
- SQL: Business questions
- Python: Analytical questions
- Power BI: 5 dashboards

## 🎯 Business Objectives

- Analyze sales performance
- Identify top products and stores
- Perform Pareto (80/20) analysis
- Detect stockout and overstock risks
- Segment stores using K-Means clustering
- Forecast future sales

## 🛠️ Tech Stack

- SQL
- Python (Pandas, Matplotlib, Scikit-learn)
- Power BI
- DAX

### Key Findings

| Best Performing Store | **Ciudad de Mexico 2** |

## 🐍 Python Analysis

### Analysis Performed
- Highest Sales Month
- Lowest Sales Month
- Inventory Days Remaining
- Rolling Daily Sales
- Stockout Detection
- Overstock Analysis
- K-Means Store Segmentation

### Key Insights
- March 2023 was highest selling month
- August 2022 lowest selling month
- **20 products** are in Immediate Stockout status.
- High-performing stores achieve the highest revenue with efficient inventory.
- Stores were segmented into High, Medium and Low performing groups.
